## 🐛 Como reportar um bug

1. Verifique se o bug já foi reportado nas [Issues](https://github.com/AdlerMrF/medguard/issues)
2. Se não encontrar, abra uma nova issue com:
   - Título claro e descritivo
   - Passos para reproduzir o problema
   - Comportamento esperado vs comportamento atual
   - Versão do projeto, SO e versão do Python

---

## 💡 Como sugerir uma melhoria

1. Abra uma [Issue](https://github.com/AdlerMrF/medguard/issues) com o título começando por `[SUGESTÃO]`
2. Descreva claramente:
   - Qual problema a melhoria resolve
   - Como você imagina a solução
   - Se possível, exemplos de uso
